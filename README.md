1. 微擎不得强制开发者收取年费,把收费权限交还给开发者;
2. 微擎不得扰乱自由交易制度;
3. 微擎不得强制停止应用使用权;用户自愿决定是否购买更新服务;
4. 微擎不得强制植入云控台代码,随时篡改用户服务器代码;
5. 微擎不得限制用户模块自由转让合并,把之前承诺的自由交易合并权交还给用户;
